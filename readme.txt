== Description ==

Attentive Security Plugin is a very simple Wordpress security plugin. With this plugin you can hide your current WordPress version from any resource, You can redirect your wp-admin link to any page. So if anyone tries to access the wp-admin link they won't be able to do that.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Attentive - WordPress security plugin' menu in WordPress.
3. Go to Settings => Attentive WSP Settings => and follow the options how you need.
